JiffyDOS kernals (not redistributable) go here.

Put 16K (16384-byte) .bin files whose names contain PAL or NTSC, e.g.:

  JiffyDOS_Plus4_6.01_PAL.bin
  JiffyDOS_Plus4_6.01_NTSC.bin

Then run the build and choose kernal option "2) JiffyDOS".

This directory is gitignored except for this README.
6510-patched kernals are downloaded automatically from hackjunk.com (see sources/6510/).
